CSS Elements

Text can be Bolded by being wrapped in strong, or include font-weight: bold; in style

Text can be Underlined by being wrapped in u, or include text-decoration: underline in style

Text can be Italicize by being wrapped in em, or include text-style: italic; in style

Text can be Stricked-Through by being wrapped in strong, or include text-decoration: line-through in style
