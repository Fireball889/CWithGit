Flex-Wrap

This enables the flex item to be split into multiple rows(or columns, depending on flex direction)

flex-wrap:

  • nowrap (default)
  • wrap
  • wrap-reverse

Flex-Shrink (applied on flex-item)

Previous properties are applied to the flex container, flex-shrink is applied to the flex item.

This controls the size of the flex-item when the size of the parent container shrinks

This property allows an item to shrink if the flex container is too small. Items shrink when the width of the parent container is smaller than the combined widths of all flex items

The higher the flex-shrink value, the more it will shrink compared to the other items in the container

flex-shrink: 1
flex-shrink: 2

Flex-Grow

This controls the size of the flex-item when the parent container expands

The higher the flex-grow value, the more it will grow

flex-grow: 1
flex-grow: 2

Flex-Basis

This can be used to specify the initial size of the item before CSS makes adjustment with flex-shrink or flex-grow

flex-basis: 10em
flex-basis: 20em

Set shrink/grow/basis in one line using flex:

flex: 1 0 10px;

This sets flex-shrink to 1, flex-grow to 0, and flex-basis to 10px;

The default flex if not specified is flex: 0 1 auto;

CSS Grid

CSS Grids help you easily build complex web designs. It works by turning an HTML element into a grid container with rows and columns for you to place children elements where you want within the grid

Useful link

display: grid

Any HTML elements can be turned into a gride by using display: grid, this gives you the ability to use all the other properties associated with CSS grid

Columns can be added using grid-template-columns: 100px 100px 100px.

Rows can be added using grid-template-rows: 50px 50px.

Using CSS Grid units to change the size of columns and rows

Absolutely unit px and relative unit em can be used, the following CSS Grid specific units can be used as well:

  • fr: Sets the column or row to a fraction of the available space
  • auto: Sets the column or row to the width or height of its content automatically
  • $: Adjusts the column or row to the percent width of its container

Column/Row Gap

grid-column-gap can be used to create a gap between the columns.

grid-row-gap can be used to create a gap between the rows.

grid-gap can be used as a more condensed way of creating gaps, where grid-gap: 10px; would create column & row gaps of 10px, and grid-gap: 10px 5px; would create row gap 10px, and column gap 5px.

Grid properties

You can control the amount of columns an item will consume by using grid-column: 1 / 3;, which will let the item start at column line 1, and stop at column line 3.

grid-row: 1 / 3; will let the item start at row line 1, and stop at row line 3.

Align An Item Horizontally

justify-self can be used to align an item horizontally, by default: justify-self: scretch;, which will make the content fill the whole width of the cell.

Other values can be:

  • start: Aligns the content at the left of the cell.
  • center: Aligns the content in the center of the cell.
  • end: Aligns the content at the right of the cell.

Align An Item Vertically

Similar to justify-self, items can be aligned vertically using align-self.

Align All Items Within Container

justify-items can be used to align all items horizontally whtin the grid container

align-items can be used to aligh all items vertically within the grid container

Using Repeat Function

When creating more grids, it can get very repetitive, repeat:(3, 50px 100px) 25px; can be used to reduce repitition.

The above code will create the following grid

50px 100px 50px 100px 50p 100px 25px

Limit item size using minmax function

minmax:(50px, 200px); defines that this element has a minimum size of 50px, and a maximum size of 200px.

This function can be used in conjunction with repeat function, for example: repeat(10, min(50px, 1fr));

Create flexible layouts using auto-fill&auto-fit

repeat(auto-fill, minmax(60px, 1fr)); will keep inserting 60px columns and stretching them until maximum height, and insert the next column

repeat(auto-fit, minmax(60px, 1fr)); will collapses those empty rows or columns and stretches your items to fit the size of the container

Uce media queries to create responsive layouts

The following piece of code defines that 3 different layouts for 3 different viewport sizes

@media (min-width: 300px){ .container{ grid-template-columns: auto 1fr; grid-template-rows: auto 1fr auto; grid-template-areas: "advert header" "advert content" "advert footer"; } } @media (min-width: 400px){ .container{ grid-template-areas: "header header" "advert content" "footer footer"; } }
